Face Recognition And Emotion Detection from Picture

This is model build from scratch and trained thus the accurracy is little less.

The model is created using open cv library, which is the most common computer vision library and is used for printing images, writing on images and for drawing on images. the whole model is trained by myself and is trained for recognising images of amir khan, angelina jolie, barack obama,donald trump, elon musk, joe Biden, Leonardo dicaprio, Robert Drowney jr, Rogerer Fedrer, Tom Cruise, Scarlett Johansson, Salman Khan, Sachin Tendulkar, Christiano Ronaldo and Lionel Messi. around 100 images of all these persons were taken and then were used to train the model. The accuracy of the model is around 70%.
For the emotion detection I have used data set on the kaggle and train using that dataset. the accuracy of this model is around 32%. Since it is one of the most difficult model and its my first model though I am not happy but I am satisfied with my accuracy. Even the top person on kaggle when I checked was having just 68% accurracy. Accuracy can be increased using more larger dataset and more powerful training tools.
